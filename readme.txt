日语学习工具集 v1.0

一、项目说明
这是一个基于网页的日语学习工具集，包含格式转换工具和数据管理功能。主要用于处理和展示日语学习相关的数据。

二、项目结构
format-converter/
├── index.html                    # 主页面（iframe容器）
├── css/
│   └── style.css                # 全局样式文件
├── js/
│   └── main.js                  # 主程序入口（菜单控制）
├── modules/                      # 功能模块目录
│   ├── format-tool/             # 格式转换工具模块
│   │   ├── index.js            # 转换功能实现
│   │   └── history.js          # 历史记录功能
│   └── data-list/              # 数据列表模块
│       ├── db-config.js        # 数据库配置
│       └── index.js            # 数据列表功能
├── pages/                       # iframe页面目录
│   ├── format-tool.html        # 格式转换工具页面
│   └── data-list.html          # 数据列表页面
└── utils/
    └── toast.js                # 全局提示工具

三、功能模块

1. 格式转换工具
功能说明：
- JSON格式转换：支持日语学习数据的特殊格式化
- 历史记录：自动保存最近50条转换记录
- 一键复制：快速复制转换结果
- 一键清空：清空输入内容

输入格式：["日语文本(中文翻译) 罗马音"]
输出格式：
{
    "japanese": "日语文本",
    "chinese": "中文翻译",
    "romaji": "罗马音"
}

2. 数据列表功能
功能说明：
- 数据展示：展示数据库中的日语句子数据
- 分页浏览：支持数据分页查看
- 音频播放：支持音频文件播放（如有）
- 响应式设计：适配各种屏幕尺寸

四、技术架构

1. 前端架构
- 基于iframe的模块化设计
- ES6+ 模块化开发
- 响应式布局
- 暗色模式支持

2. 后端架构
- MySQL数据库（8.0+）
- 数据库配置：
  主机：localhost
  端口：3306
  用户：root
  数据库：api_test_db

3. 数据结构
japanese_sentences表结构：
- id: int (自增主键)
- japanese: varchar(255) (日语原文)
- chinese: varchar(255) (中文翻译)
- romaji: varchar(255) (罗马音)
- audio_url: varchar(512) (音频文件URL)
- created_at: timestamp
- updated_at: timestamp

五、设计规范

1. 颜色方案
- 主色调：#84B1ED（天蓝色）
- 次要色：#FFB7B7（粉红色）
- 强调色：#C7B7ED（淡紫色）
- 背景色：#F0F8FF（淡蓝色）
- 文本色：#4A4A4A（深灰色）

2. 布局设计
- 固定宽度侧边栏（200px）
- 自适应内容区域
- 响应式设计断点：
  移动端：<768px
  平板：768px-1024px
  桌面：>1024px

六、开发规范

1. 代码规范
- 使用ES6+语法
- 模块化开发
- 类名使用PascalCase
- 方法名使用camelCase
- 常量使用UPPER_SNAKE_CASE

2. 文件组织
- 按功能模块划分目录
- 使用小写字母命名文件
- 使用连字符分隔单词
- 保持目录结构清晰

3. 注释规范
- 必须为类和方法添加注释
- 复杂逻辑需要详细注释
- 使用英文编写注释
- 保持注释及时更新
